# Руководство по установке InfinityX
> [!Warning]
> Данные инструкции предназначены для InfinityX на Xiaomi Pad 5 (nabu). Они сработают только если вы точно выполните каждый раздел и шаг.

## Подготовка к установке

Скачайте следующие файлы из [```Релизов```](https://github.com/hxfuxyy/infota/releases/):
- **InfinityX.zip** — файл прошивки для ADB sideload
- **boot.img**, **vendor_boot.img**, **dtbo.img** — нужны только для **Варианта А (ADB Sideload)**

Если вы планируете использовать **Авто Установщик** (установка через PC-скрипты), вам нужно только скачать его из **закреплённого сообщения** в Telegram-чате [t.me/InfinityXnabu](https://t.me/InfinityXnabu) — больше ничего не требуется.

## Шаг 1: Прошивка Recovery

> [!Note]
> Этот шаг нужен только для **Варианта А (ADB Sideload)**. Если вы планируете использовать **Авто Установщик**, пропустите сразу до [Варианта Б](#вариант-б-авто-установщик-pc-скрипты).
> Этот шаг также можно пропустить, если у вас уже установлен совместимый кастомный рекавери.

1. На компьютере откройте командную строку (в Windows) или терминал (в Linux или macOS) и введите:
```cmd
adb -d reboot bootloader
```
> Также можно выключить устройство и загрузиться в режим bootloader, удерживая **Уменьшение громкости + Питание**.

2. Когда устройство перейдёт в режим fastboot, убедитесь что ПК его видит:
```cmd
fastboot devices
```

3. Прошейте скачанные файлы образов на устройство:
```cmd
fastboot flash boot boot.img
fastboot flash vendor_boot vendor_boot.img
fastboot flash dtbo dtbo.img
```

4. Перезагрузитесь в рекавери чтобы проверить установку:
```cmd
fastboot reboot recovery
```

> [!Warning]
> **Не** перезагружайтесь в существующую ОС — это перезапишет только что установленный рекавери.

## Шаг 2: Установка прошивки

Выберите один из двух методов ниже.

### Вариант А: ADB Sideload

> [!Note]
> Этот вариант также можно использовать для обновления прошивки.

**Только для чистой / первой установки:**
- Нажмите **Factory Reset > Format data** и продолжите процесс форматирования.
- Это удалит шифрование и все файлы во внутреннем хранилище.
- Этот шаг можно **пропустить** при обновлении прошивки.

1. На устройстве нажмите **Apply Update > Apply from ADB**, чтобы начать sideload.

2. На компьютере выполните команду:
```cmd
adb -d sideload InfinityX.zip
```

3. После установки рекавери может предложить перезагрузиться обратно для дополнительных пакетов — выберите **«No»** и нажмите **Reboot to system**.

## Обновление прошивки

Если вы обновляетесь до новой версии, сброс до заводских настроек **не требуется**.

- **Через OTA:** Используйте встроенный Updater — *Настройки > Система > Обновление*
- **Через Recovery:** Следуйте [Варианту А](#вариант-а-adb-sideload) выше без выполнения Factory Reset.
- **Через Авто Установщик:** Запустите скрипт `update_...` из ZIP-архива Авто Установщика.

> [!Warning]
> Всегда проверяйте, что ZIP или файл обновления соответствует вашей текущей версии прошивки. Установка несовместимой версии может вызвать boot loop или необратимые повреждения.
