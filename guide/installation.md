# Гайд з встановлення InfinityX
> [!Warning]
> Наведені інструкції призначені для InfinityX на Xiaomi Pad 5 (nabu). Вони спрацюють лише якщо ви точно виконаєте кожен розділ та крок.

## Попередня підготовка

Завантажте наступні файли з [```Релізів```](https://github.com/hxfuxyy/infota/releases/):
- **InfinityX.zip** — файл прошивки для ADB sideload
- **boot.img**, **vendor_boot.img**, **dtbo.img** — потрібні лише для **Варіанту А (ADB Sideload)**

Якщо ви плануєте використовувати **Авто Інсталятор** (встановлення через PC-скрипти), вам потрібно лише завантажити його з **закріпленого повідомлення** у Telegram-чаті [t.me/InfinityXnabu](https://t.me/InfinityXnabu) — більше нічого не потрібно.

## Крок 1: Прошивка Recovery

> [!Note]
> Цей крок потрібен лише для **Варіанту А (ADB Sideload)**. Якщо ви плануєте використовувати **Авто Інсталятор**, пропустіть одразу до [Варіанту Б](#варіант-б-авто-інсталятор-pc-скрипти).
> Цей крок також можна пропустити, якщо у вас вже встановлено сумісний кастомний рекавері.

1. На комп'ютері відкрийте командний рядок (у Windows) або термінал (у Linux або macOS) і введіть:
```cmd
adb -d reboot bootloader
```
> Також можна вимкнути пристрій і завантажитися в режим bootloader, утримуючи **Зменшення гучності + Живлення**.

2. Коли пристрій перейде в режим fastboot, переконайтеся, що ПК його бачить:
```cmd
fastboot devices
```

3. Прошийте завантажені файли образів на пристрій:
```cmd
fastboot flash boot boot.img
fastboot flash vendor_boot vendor_boot.img
fastboot flash dtbo dtbo.img
```

4. Перезавантажтеся в рекавері, щоб перевірити встановлення:
```cmd
fastboot reboot recovery
```

> [!Warning]
> **Не** перезавантажуйтеся в існуючу ОС — це перезапише щойно встановлений рекавері.

## Крок 2: Встановлення прошивки

Оберіть один із двох методів нижче.

### Варіант А: ADB Sideload

> [!Note]
> Цей варіант також можна використовувати для оновлення прошивки.

**Тільки для чистого / першого встановлення:**
- Натисніть **Factory Reset > Format data** і продовжіть процес форматування.
- Це видалить шифрування та всі файли у внутрішньому сховищі.
- Цей крок можна **пропустити** при оновленні прошивки.

1. На пристрої натисніть **Apply Update > Apply from ADB**, щоб розпочати sideload.

2. На комп'ютері виконайте команду:
```cmd
adb -d sideload InfinityX.zip
```

3. Після встановлення рекавері може запропонувати перезавантажитися назад для додаткових пакетів — оберіть **«No»** і натисніть **Reboot to system**.

### Варіант Б: Авто Інсталятор (PC-скрипти)

> [!Note]
> Завантажте ZIP з Авто Інсталятором із **закріпленого повідомлення** у Telegram-чаті [t.me/InfinityXnabu](https://t.me/InfinityXnabu).

<details>
<summary>Windows</summary>

> Використовуйте цей метод лише якщо ви вмієте підключати планшет до комп'ютера та запускати скрипти. Неправильне використання Fastboot може видалити системні розділи.

1. **Розпакуйте** ZIP з Авто Інсталятором у зручне для вас місце.
   - Уникайте шляхів з пробілами або спецсимволами, щоб запобігти помилкам у скриптах.

2. Підключіть планшет у режимі **Fastboot**:
```cmd
adb -d reboot bootloader
```
> Або вимкніть пристрій і утримуйте **Зменшення гучності + Живлення** до появи екрану завантажувача.

3. Переконайтеся, що пристрій визначається:
```cmd
fastboot devices
```
> Якщо пристрій не з'являється — перевірте USB-кабель, спробуйте інший роз'єм або встановіть відповідні драйвери.

4. Запустіть відповідний скрипт із розпакованої папки:
   - `install_..._windows.bat` — для першого встановлення
   - `update_..._windows.bat` — для оновлення існуючої прошивки

- Уважно дотримуйтеся інструкцій на екрані.

> [!Warning]
> **Не від'єднуйте планшет під час виконання скрипту.** Переривання може зробити пристрій незавантажуваним.

</details>

```cmd
adb -d reboot bootloader
```


> **Не закривайте термінал і стежте за процесом.** Дострокове закриття терміналу може перервати прошивку.

> Перше завантаження може тривати **5–10 хвилин**. Не панікуйте і не переривайте процес.

## Оновлення прошивки

Якщо ви оновлюєтесь до нової версії, скидання до заводських налаштувань **не потрібне**.

- **Через OTA:** Використовуйте вбудований Updater — *Налаштування > Система > Оновлення*
- **Через Recovery:** Дотримуйтесь [Варіанту А](#варіант-а-adb-sideload) вище без виконання Factory Reset.
- **Через Авто Інсталятор:** Запустіть скрипт `update_...` із ZIP-архіву Авто Інсталятора.

> [!Warning]
> Завжди перевіряйте, що ZIP або файл оновлення відповідає вашій поточній версії прошивки. Встановлення несумісної версії може спричинити boot loop або незворотні пошкодження.
